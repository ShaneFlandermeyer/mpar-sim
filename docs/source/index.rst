Welcome to Lumache's documentation!
===================================

**mpar-sim** is a Python library that contains general radar signal processing tools and basic detection-level radar data simulation. At the moment, the library primarily contains tools for tracking and array resource management. However, the library is under active development and will be expanded to include more general radar signal processing tools.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. .. note::

..    This project is under active development.
